# 山东疯狂 8 点 (Crazy Eights)

一款基于 React + Vite + Tailwind CSS 构建的经典扑克牌游戏。

## 游戏特色
- **经典规则**：匹配花色或点数，8 是万能牌。
- **AI 对手**：内置智能 AI 陪玩。
- **中文语音**：支持游戏过程中的中文语音播报。
- **响应式设计**：完美适配手机、平板和电脑。
- **个性化背景**：极具特色的“肥猪”主题背景。

## 核心规则
1. **发牌**：每人初始 8 张牌。
2. **出牌**：出的牌必须在“花色”或“点数”上与弃牌堆顶部的牌匹配。
3. **万能 8 点**：数字“8”是万用牌，可以在任何时候打出，并指定新的花色。
4. **摸牌**：无牌可出时必须摸一张牌。
5. **胜负**：最先清空手牌的一方获胜。

## 开发与部署

### 本地运行
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`

### 部署到 Vercel
本项目已配置好 `vercel.json`，直接关联 GitHub 仓库即可一键部署。

## 技术栈
- React 19
- Vite 6
- Tailwind CSS 4
- Lucide React (图标)
- Motion (动画)
